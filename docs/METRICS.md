1. Metric 1: Net Promoter Score

Implemented by embedding a survey from the FlutterFlow marketplace in SmartFuel and prompting the user to answer after three logins via a custom function. Data will be embeded in the user database for that user. Results will be analyzed to gauge overall user satisfaction.

2. Metric 2: Adoption

First_open: 

* Built-in event in Firebase.
* Tracks when someone opens the app for the first time.

Open Firebase Console.
Go to Analytics → Events.
Look for the event first_open.

3. Metric 3: Retention

User_retention: 

* Measures the percentage of users who return to the app after their first visit.

Open Firebase Console.
Click Analytics → Retention.
Firebase will automatically show:
Day 1 retention
Day 7 retention
Day 30 retention

4. Metric 4: DAU

Measures the number of unique users who open and use the SmartFuel app each day. This metric helps determine how frequently users are engaging with the application. To access, open Firebase Console, navigate to Analytics → Dashboard, then view Daily Active Users (DAU) under the Engagement metrics.
